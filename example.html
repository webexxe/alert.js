ALERT JS ÖRNEKLERİ

$.alertJS Kullanımı

$.alertJS() fonksiyonu ile masıjımızı uyarı ekranında full veya mini olarak gösterebilirsiniz.

Aldığı değerler : $.alertJS(mode, message, status, close, id)

$.alertJS("mini", "Benim Mesajım", "success", 1, 5)
mode      / string    -   default = "full"                -   "full", "mini"
message   / string    -   default = "Message Not Found!"  -
status    / string    -   default = "info"                -   "info", "error", "success", "alert"
close     / number    -   default = 1                     -   0 = close function, 1 = close button, 2 = close button and mask
close     / number    -   default = 1                     -   0 = close function, 1 = close button, 300 = auto close milisecond
id        / number    -   default = Random 0 - 10         -

$.alertJSLoad Kullanımı

$.alertJSLoad() fonksiyonu ile yükleme sırasında ekrana uyarı mesajı verebilirsiniz. Uyarıyı kapatmak için yapılan işlemden sonra $.alertJSX() fonksiyonunu kullanmanız gerekir.

Aldığı değerler : $.alertJSLoad(mode, message, id, loadFigur, status)

$.alertJSLoad("full", "Yükleniyor...", 5, "refresh", "info")
mode      / string    -   default = "full"                -   "full", "mini"
message   / string    -   default = "Message Not Found!"  -
id        / number    -   default = Random 0 - 10         -
loadFigur / string    -   default = "refresh"             -   "spinner", "circle-o-notch", "refresh", "cog"
status    / string    -   default = "info"                -   "info", "error", "success", "alert"

$.alertJSX Kullanımı

$.alertJSX() fonksiyonu ile uyarılarınızı işlem bittiğinde kapatabilirsiniz.

Aldığı değerler : $.alertJSX(id)

$.alertJSX(5)
id        / number    -   target id number
